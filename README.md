# content_recommender

Recommendation system based on advanced data driven technics (NN, interaction matrix?TF-IDF ...)

## Technology stack (Python)

Core libraries (see `pyproject.toml` / `uv.lock` for pinned versions):

- **pandas**, **polars** — tabular data
- **numpy** — numerical arrays
- **matplotlib**, **seaborn** — plotting
- **scikit-learn** — machine learning
- **jupyter** — notebooks (JupyterLab / classic notebook via the `jupyter` metapackage)
